YouTube Favorite Exporter

A simple application built using the Google data APIs which connects to YouTube and downloads all of your favorites into a file. Uses the Netscape Bookmark File Format. Can be imported into Chrome, Firefox, IE, Delicious etc. Requires .NET Framework 2.0.

When complete, a temporary bookmark file is created and opened in your browser
